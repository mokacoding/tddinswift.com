# `tddinswift.com`

This repository is the source for [tddinswift.com](https://tddinswift.com).

## Commands

- `npm install`
  Install the site dependencies.
- `hugo server`
  Serve the site locally with live reload.
- `hugo`
  Build the production site into `public/`.
  There is no separate deploy command documented in this repository, `git push` and GitHub pages will take over
